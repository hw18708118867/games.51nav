## About the Sequence Memory Test

Sequence memory is a fundamental cognitive ability that underlies many everyday activities. From remembering a phone number long enough to dial it, to following a recipe step by step, sequence memory is constantly at work.

### The Science of Sequential Memory

Sequential memory relies on the phonological loop and visuospatial sketchpad — two components of working memory identified by cognitive psychologists. The phonological loop handles verbal sequences (like numbers or words), while the visuospatial sketchpad handles spatial sequences (like positions or movements).

### How This Test Challenges You

Our Sequence Memory Test combines visual and spatial memory. You watch tiles light up in a specific order, then must reproduce that order. As you succeed, the sequence grows longer, pushing the limits of your working memory capacity. The test ends when you make a mistake, giving you a clear measure of your current capacity.

### Real-World Applications

Strong sequence memory helps with learning new skills, following multi-step instructions, and even social interactions (remembering the flow of a conversation). Musicians, dancers, and athletes often have exceptional sequence memory from years of practice.
