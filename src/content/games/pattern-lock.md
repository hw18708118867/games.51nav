## About Pattern Lock

Pattern lock memory is inspired by the unlock patterns used on Android phones. The 3×3 grid of dots creates a familiar interface that's instantly recognizable. But instead of using it as a security measure, we use it as a memory training tool.

### The Memory Challenge

Remembering a sequence of positions combines visual-spatial memory with sequential memory. The 3×3 grid provides a natural structure that helps with chunking — grouping dots into recognizable shapes. As the pattern grows longer, it pushes the limits of working memory, making this an effective cognitive training exercise.
