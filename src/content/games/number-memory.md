## About the Number Memory Test

The Number Memory Test is based on the digit span task, one of the oldest and most widely used measures of short-term memory in cognitive psychology. First developed in the 19th century, digit span testing remains a core component of modern intelligence and memory assessments.

### The Psychology Behind Digit Span

Your short-term memory (also called working memory) is like a mental workspace where you temporarily hold information. For most people, this workspace can hold about 7 items at once. When you try to remember a sequence of digits, each digit takes up one "slot" in this workspace.

As the number gets longer, it exceeds your natural capacity, and you need to use strategies like chunking to remember more. Chunking involves grouping individual digits into larger, more meaningful units. For example, instead of remembering "1-4-9-2" as four separate digits, you might remember "1492" as the year Columbus sailed to America.

### How Our Test Works

Our test starts with a 3-digit number and increases by one digit each time you answer correctly. You have a limited time to view the number — longer numbers get more viewing time, but the challenge increases as the number grows.

The number disappears automatically, and you must type it from memory. There's no time limit for typing your answer, so you can focus entirely on accurate recall.

### What Your Results Mean

- **3-5 digits**: Below average short-term memory span
- **6-8 digits**: Average — in line with most people
- **9-12 digits**: Above average — good working memory
- **13-16 digits**: Excellent — exceptional short-term memory
- **17+ digits**: Superior — world-class digit memory

### The Real-World Importance of Digit Memory

While remembering long strings of numbers might seem like a party trick, digit span is actually correlated with many important cognitive abilities. Research has shown that working memory capacity (measured by tests like digit span) predicts academic achievement, language learning ability, and even job performance in complex roles.
