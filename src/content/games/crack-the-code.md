## About Crack the Code

Crack the Code is based on Mastermind, the classic code-breaking board game invented in 1970 by Mordecai Meirowitz. Mastermind sold over 50 million copies and was named Game of the Year multiple times. The game's combination of logic and deduction made it an instant classic.

### Deductive Reasoning

Crack the Code exercises pure deductive reasoning. Each guess provides partial information — which colors are correct and whether they're in the right position. The challenge is to combine clues from multiple guesses to narrow down the possibilities. This is the same type of reasoning used in scientific inquiry, debugging, and detective work.
